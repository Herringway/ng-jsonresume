ng-jsonresume

With ng-jsonresume you can include a resume in your website by using the <resume> tag.

Example: the following code produces the result below - take a look at the source of this page to see how to use it:
<resume dataurl="https://rawgit.com/marko-knoebl/ng-jsonresume/master/example-resume.json"></resume>

For now ng-jsonresume will use the flat theme from jsonresume. Other themes may become available in the future
